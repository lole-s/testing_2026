# Testing 2026

Repositorio de la materia **Testing** para el ciclo lectivo 2026 (estudiantes de 17 años, escuela secundaria con orientación en programación).

- Repositorio base: `https://github.com/lole-s/testing_2026`
- Contacto profesor: `jcsodo@escuelasproa.edu.ar`

## Objetivo
Aprender a **probar software**, conceptos básicos de **redes de datos**, documentar y trabajar en equipo.

## Ejes y contenidos
- **Eje 1: Software colaborativo**
  - organización de carpetas, control de versiones, documentación
- **Eje 2: Redes de datos**
  - conceptos básicos sobre redes TCP/IP
- **Eje 3: Testing**
  - casos de prueba, evidencia, reporte de errores, mejora continua

## Cómo se evalúa
- Asistencia > 70%
- Trabajo en clase y participación activa
- Entregas y calidad de evidencias
- 2 pruebas en el año / 2 recuperatorios (uno por cada prueba) (nota >= 7)
- Responsabilidad, comportamiento y conducta
- Proyecto redes de datos (nota >= 7)
