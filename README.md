# Testing 2026

Repositorio de la materia **Testing** para el ciclo lectivo 2026 (estudiantes de 17 años, escuela secundaria con orientacion en programacion).

- Repositorio base: `https://github.com/lole-s/testing_2026`
- Contacto profesor: `jcsodo@escuelasproa.edu.ar`

## Objetivo
Aprender a **probar software**, conceptos basicos de **redes de datos**, documentar y trabajar en equipo.

## Ejes y contenidos
- **Eje 1: Software colaborativo**
  - organizacion de carpetas, control de versiones, documentacion
- **Eje 2: Redes de datos**
  - conceptos basicos sobre redes TCP/IP
- **Eje 3: Testing**
  - casos de prueba, evidencia, reporte de errores, mejora continua

## Como se evalua
- Asistencia > 70%
- Trabajo en clase y participacion activa
- Entregas y calidad de evidencias
- 2 pruebas en el ano / 2 recuperatorios (uno por cada prueba) (nota >= 7)
- Responsabilidad, comportamiento y conducta
- Proyecto redes de datos (nota >= 7)
