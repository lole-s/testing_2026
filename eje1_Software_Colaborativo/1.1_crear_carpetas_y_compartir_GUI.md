# 1.1 Crear carpetas y compartir (GUI + linea de comandos)

## Objetivo
Organizar carpetas del curso, practicar comandos basicos de Windows y compartir la carpeta en Google Drive.

## Parte A: GUI (Explorador de archivos)
1. Crear carpeta local: `C:\temp2026\Testing_APELLIDO`.
2. Dentro de `Testing_APELLIDO` crear la subcarpeta `eje1_Software_Colaborativo`.
3. Subir la carpeta `Testing_APELLIDO` a Google Drive institucional.
4. Compartir el enlace de la carpeta con `jcsodo@escuelasproa.edu.ar`.

## Parte B: Linea de comandos (PowerShell o CMD)

### Comandos basicos (equivalencias)
- `cd` cambia de carpeta.
- `mkdir` crea carpeta.
- `dir` (CMD) o `ls`/`Get-ChildItem` (PowerShell) lista archivos.
- `type` (CMD) o `Get-Content` (PowerShell) muestra contenido.

### Ejemplo guiado (PowerShell)
```powershell
cd C:\
mkdir temp2026
cd temp2026
mkdir Testing_APELLIDO
cd Testing_APELLIDO
mkdir eje1_Software_Colaborativo
ls
```

### Ejemplo guiado (CMD)
```bat
cd C:\
mkdir temp2026
cd temp2026
mkdir Testing_APELLIDO
cd Testing_APELLIDO
mkdir eje1_Software_Colaborativo
dir
```

## Diferencias clave entre PowerShell y CMD
- PowerShell usa objetos y cmdlets (por ejemplo `Get-ChildItem`), CMD usa comandos tradicionales (`dir`).
- PowerShell acepta `ls` y `dir` como alias, pero su salida es mas rica.
- CMD es mas simple; PowerShell es mas potente para automatizar tareas.
