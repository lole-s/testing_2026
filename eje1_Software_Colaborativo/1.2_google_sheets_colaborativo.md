# 1.2 Google Sheets colaborativo

## Objetivo
Crear un archivo compartido y trabajar en simultaneo en un mismo documento.

## Consigna
1. Crear un archivo de Google Sheets llamado `testing2026` dentro de la carpeta compartida del curso.
2. Compartir el archivo con 3 estudiantes (editar):
   - Estudiante A
   - Estudiante B
   - Estudiante C
3. Trabajo simultaneo:
   - Cada estudiante crea una hoja con su nombre.
   - Cada uno completa 5 filas con datos de ejemplo (fecha, tema, actividad).

## Practica de funciones basicas
- Arrastrar y soltar para copiar una serie (autorrelleno).
- Remarcar una tabla o rango de celdas y aplicar formato (bordes o color).
- Ordenar una columna de manera ascendente.

## Entrega
- Enlace compartido y una captura donde se vea que las 3 hojas fueron creadas.
